# 善果佛化禮儀 官方網站（Stitch 拼貼畫布版）

依 Stitch 概念稿的「Compassionate Transition．拼貼畫布」設計語言重新製作的一版，純 HTML / CSS / JS，雙擊任一 `.html` 即可瀏覽，無需後端。

## 設計特色

1. **拼貼畫布風格**：宣紙米白底（#FDFBF7）加紙纖維紋理，菩提暖金（#D4AF37）、安定青綠（#2E5A56）、夕陽柔桃（#F2997B）三色系，柔邊花瓣遮罩、浮動光暈、層疊菩薩插圖，營造手作畫布的溫度。
2. **字型**：標題思源宋體、內文思源黑體（Google Fonts，需連網載入；離線時退回系統字型）。
3. **互動感**：
   - 導覽列捲動時加深玻璃模糊感。
   - 全站區塊捲動進場（fade-up）。
   - 西方三聖卡片可輕觸翻面，閱讀菩薩的願。
   - FAQ 手風琴（一次只開一個），常見問題頁另有分類篩選。
   - 方案報價頁有即時費用試算機（選方案、勾規費，自動加總 A＋B）。
   - 聯絡表單前端送出示意。
4. **圖片**：全部採用客戶自製素材。西方三聖角色插圖、阿彌陀佛無量光接引橫幅、善果 LOGO、慧開法師照片。
5. **RWD**：手機版底部固定「撥打／加 LINE」雙鍵，桌機右下浮動 LINE。
