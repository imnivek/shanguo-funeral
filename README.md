## 設計

宣紙米白底配菩提暖金、安定青綠、夕陽柔桃，搭配自製西方三聖角色插圖與蓮花意象的「拼貼畫布」風格。標題思源宋體、內文思源黑體。全站 RWD 響應式，手機底部固定撥打／LINE 雙鍵。
